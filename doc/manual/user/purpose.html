SqlParser - Users' Manual - Purpose

Purpose of the Project

SqlParser was developed as part of the SIARD project. It is used by all JDBC wrapper implementations of the SIARD project in order to achieve a common truly standardized access to various proprietary databases. This is achieved by extending the SqlFactory to create selected extended SqlParser classes which convert standard SQL:2008 (parsed using parse()) to proprietary DBMS SQL (overriding format() as needed).

Another use was made of the evaluation feature of SqlParser (currently limited to single-table queries) in the JDBC implementation for MS Access in the project JdbcAccess.

A quite general parser for SQL:2008 can be useful for many other purposes without any relationship with the SIARD project. Therefore it is published as a separate stand-alone project.
